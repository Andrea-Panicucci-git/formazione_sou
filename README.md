## Installazione

Per eseguire questo progetto, segui i passaggi sottostanti:

    ```bash
    git clone https://github.com/tuousername/progetto.git
    ```
2. Naviga nella directory del progetto:
    ```bash

    ```
3. Installa le dipendenze:

        ```bash
        npm install
        ```
    - Se utilizzi **pip**:

        ```bash
        pip install -r requirements.txt
        ```
